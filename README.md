Vector Norm: 𝑎2 + 𝑏2 + 𝑐2
Distance formula: (𝑎1−𝑎2)2 + (𝑏1−𝑏2)2 + (𝑐1−𝑐2)2
Dot Product: (a1 , b1 , c1) . (a2 , b2 , c3) = a1*b2 + b1 * b2 + c1 * c3
Example: (2, 3, 4) . (5, 6, 7) = 10 + 18 + 28 = 56
